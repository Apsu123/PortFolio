Robo Farm – Automated Garden Watering System

Robo Farm is an Arduino-based project aimed at automating small-scale irrigation systems. Using a relay and submersible pump, the board activates water flow at fixed intervals, demonstrated every 10 seconds in the urban prototype. Power optimization is achieved through an Arduino sleep-mode library that minimizes energy usage between watering cycles, allowing long-term autonomous operation. The system is scalable to full garden setups with stronger pumps and sensor-based moisture control in future iterations. This project showcases practical hardware programming and real-world IoT application design.
