Smart Safe – Programmable Electronic Lock

Smart Safe is a DIY project to retrofit an old safe with an Arduino-controlled digital number lock. The system allows users to open and lock the safe using a numeric keypad and includes the ability to change the access code at any time. The lock mechanism is powered by a small DC motor, controlled via digital pin outputs on the Arduino Uno board. The project demonstrates basic principles of embedded systems, including input reading, motor control, and programmatic logic for code validation. Future improvements include adding an LCD display for feedback and EEPROM storage to retain the access code after power cycles.
