LearningHub – Interactive AI-Powered Language Learning

LearningHub is an interactive web application designed to enhance language learning by making every new word in a text clickable. When clicked, words are processed through an AI chatbot API that provides translations and contextual meanings in real time. The project began with simple wireframes and evolved into a fully functional demo with live AI integration. Throughout development, the focus remained on usability, clean interface design, and efficient debugging of text parsing. Future possibilities include expanding the functionality to other contexts, such as recipe guidance or technical documentation reading.
